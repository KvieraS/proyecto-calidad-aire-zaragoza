# Proyecto Final - Calidad del aire en Zaragoza y su relación con la meteorología

## Objetivo
Analizar la evolución de la calidad del aire en Zaragoza y estudiar su relación con distintas variables meteorológicas.

## Fuentes de datos
- Datos abiertos del Ayuntamiento de Zaragoza
- AEMET OpenData

## Herramientas utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Visual Studio Code
- Jupyter Notebook
- Power BI

## Estructura del proyecto
- `data/raw/`: datos en bruto
- `data/processed/`: datos transformados
- `notebooks/`: notebooks del análisis
- `src/`: funciones auxiliares
- `reports/`: informe e imágenes
- `dashboard/`: archivo Power BI

Estructura inicial creada. Pendiente de descarga, limpieza, unión y análisis de los datasets.
